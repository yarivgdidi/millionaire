# Millionaire

###An Angular implementation for "Who wants to be a millionaire"
#####development [assignment](./ZoomInfo_Development_Assignment.pdf) from Zoominfo


## Running the project
```
git clone https://github.com/yarivgdidi/millionaire.git
cd millionaire
npm i
ng serve
http://localhost:4200/

```
